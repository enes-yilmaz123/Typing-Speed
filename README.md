# ⌨️ TypingPeak

TypingPeak is a modern web application designed to help users improve their typing speed and accuracy through real-time typing tests and interactive lessons.

## 🚀 Features

### 🎯 Typing Tests
- Multiple test durations:
  - 15 seconds
  - 30 seconds
  - 60 seconds
  - 120 seconds
- Real-time WPM (Words Per Minute) calculation
- Standard WPM calculation
- Accuracy tracking
- Instant performance feedback

### 📚 Lessons
- Interactive typing lessons
- Progressive difficulty levels
- Practice-oriented learning experience
- Skill improvement tracking

### 📊 Statistics
- Live typing statistics
- Accuracy percentage
- Typing speed analysis
- Performance monitoring

### 📱 Responsive Design
- Mobile-friendly interface
- Tablet support
- Desktop optimization
- Modern and clean UI

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/typingpeak.git
```

Navigate to the project directory:

```bash
cd typingpeak
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and visit:

```text
http://localhost:3000
```

## 🎮 How to Use

1. Open the website.
2. Select a test duration.
3. Start typing the displayed text.
4. Monitor your:
   - Live WPM
   - Standard WPM
   - Accuracy
5. Complete the test and review your results.
6. Practice regularly using the lessons section.
